# Aekamber Solutions Website

GitHub Pages friendly static website.

## Files
- `index.html` - Home page
- `services.html` - Services page
- `about.html` - About page
- `contact.html` - Contact page
- `styles.css` - Website styling
- `script.js` - Website interactions
- `images/` - Logo assets

## Publish on GitHub Pages
1. Upload all files in this folder to the root of your GitHub repository.
2. Go to **Settings → Pages**.
3. Source: **Deploy from a branch**.
4. Branch: **main** and folder: **/root**.
5. Save and wait 1–3 minutes.

Do not upload the outer ZIP as a single file. Extract it first and upload the contents.
